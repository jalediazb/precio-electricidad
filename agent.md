# Agents.md

## Code style
- Javascript vanilla standard.
- Todo el texto que se muestra al visitante debe estar en español.

## Herramientas
- Usa el framework Bulma CSS exclusivamente. Utilizalo siempre sus clases para el diseño. Si en alguna ocasión es necesario algo adicional, pide permiso.
- Esta web utiliza eleventy. NO añadas frameworks adicionales sin permiso.
- Siempre es preferible el uso de javascript vainilla. Cuando no sea eficiente y la instalación de paquetes a través de npm sea necesario, pide permiso.
