# Que es el mercado regulado de la electricidad en Espana

El mercado regulado de la electricidad para hogares se conoce como **PVPC** (Precio Voluntario para el Pequeno Consumidor). Es una tarifa en la que el precio de la energia se actualiza por horas y se basa en criterios oficiales y publicos.

Este modelo esta pensado para consumidores domesticos con una potencia contratada habitual de vivienda. A diferencia de muchas tarifas de mercado libre, aqui no hay un precio fijo anual pactado, sino un precio variable segun el coste real de cada hora.

En Espana, el PVPC se comercializa a traves de las **comercializadoras de referencia**, que son las empresas autorizadas para ofrecer esta modalidad regulada.

## Puntos importantes

- El precio cambia cada hora del dia.
- La metodologia de calculo es publica y supervisada.
- Permite acceder al bono social electrico, si cumples requisitos.
